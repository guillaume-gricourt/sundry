# Install

## vim-plug
```sh
# from https://github.com/junegunn/vim-plug
curl -fLo ~/.vim/autoload/plug.vim --create-dirs https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```

## Plugin
With PlugInstall:
```sh
vim ~/.vimrc
:PlugInstall
```
Standalone required:
```sh
# clang-format
brew install clang-format
```
